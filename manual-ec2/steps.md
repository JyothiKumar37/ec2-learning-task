# EC2 Creation Using AWS Console (Manual Method)

## Objective
To understand how to manually create an EC2 instance using the AWS Management Console and learn core EC2 concepts.

---

## Steps Followed

1. Logged in to AWS Management Console
2. Navigated to **Services â†’ EC2**
3. Clicked on **Launch Instance**
4. Selected **Amazon Linux 2 AMI**
5. Chose instance type **t2.micro** (Free Tier eligible)
6. Created or selected an existing **Key Pair**
7. Configured **Security Group**
   - Allowed SSH (Port 22)
8. Launched the EC2 instance

---

## Key Concepts Learned

- What is EC2
- What is an AMI
- Instance types and pricing
- Importance of Key Pairs
- Security Groups as virtual firewalls
- Difference between public and private IP

---

## Limitations of Manual Approach

- Time-consuming
- Error-prone
- Not reusable
- No version control

---

## Conclusion

Manual EC2 creation is good for learning AWS basics but not suitable for large-scale or production environments.
